# Project Apollo Case Study

## Executive Summary
We reduced customer ticket response time by 40% using the new NLP chatbot.

## Challenges
- High volume of repetitive tickets.
- Limited customer support staff.

## Implementation Details
We implemented RAG using LangChain and a ChromaDB vector store.
The knowledge base consists of 150+ markdown files.

## Outcomes
1. **Efficiency**: 40% reduction in response times.
2. **Satisfaction**: CSAT score increased to 92%.
